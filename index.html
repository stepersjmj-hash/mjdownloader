💡 로컬 사용 안내 이 파일은 브라우저에서 직접 열어 사용합니다. CORS 제한으로 인해 일부 환경에서는 브라우저를 --disable-web-security 옵션으로 실행하거나, VS Code의 Live Server 등으로 서비스해야 할 수 있습니다. 또는 Chrome 확장 프로그램 없이 직접 sssinstagram.com에서 사용하는 것과 동일한 API를 활용합니다.

사용 방법

1
인스타그램 앱/웹에서 원하는 게시물을 열고 링크를 복사합니다
2
위 입력창에 링크를 붙여넣고 [다운로드] 버튼을 클릭합니다
3
결과에서 원하는 품질을 선택하여 다운로드합니다
